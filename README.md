> A plugin for [Klasa](https://github.com/dirigeants/klasa) that deletes invites

## Install

```sh
$ npm install klasa-plugin-invites
```

## Usage

```js
const { Client } = require('klasa');

Client.use(require('klasa-plugin-invites'));
```
